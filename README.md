# WIF2003_Eco_Travel


how to run

cd project (make sure ur terminal is in same folder as docker-compose file basically)

``docker-compose up --build``

``docker-compose up --build -d`` (detached)


how to stop

``docker-compose down`` 

how to reset data 

``docker-compose down -v`` (deletes volumes)

goto website:

``localhost:3000``
